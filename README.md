# SocialFeed

**SocialFeed** —iOS-приложение на **UIKit**, которое загружает данные из API, сохраняет их в **Core Data**, поддерживает оффлайн-режим и отображает посты в виде ленты (UITableView).

Проект создан как демонстрация навыков работы с сетью, архитектурой MVC, автолэйаутами, а также локальным хранением данных.

## Архитектура

Проект построен на **MVC (Model–View–Controller)** — стандартной архитектуре Apple.

### **Model**
- `Post` — модель данных для API (DTO)
- `Entity` — Core Data сущность
- `PostLoader` — загрузка постов через сеть
- `CoreDataManager` — управление Core Data (сохранение, извлечение)

### **View**
- `PostCell` — кастомная табличная ячейка
- UIKit компоненты (UILabel, UIImageView, UITableView)

### **Controller**
- `MainFeedController`
  - загружает данные из API
  - обновляет UI
  - сохраняет и загружает данные из Core Data
  - реализует Pull-to-Refresh

---

## Функциональность

- Загрузка постов с сервера  
  API: `https://jsonplaceholder.typicode.com/posts`
- Парсинг JSON
- Кеширование данных в Core Data
- Оффлайн-режим: при отсутствии сети отображаются сохранённые посты
- Pull-to-Refresh (обновление вниз)
- Уникальные аватарки для каждого поста:  
  `https://picsum.photos/seed/{id}/80/80`
- Автоматическая высота ячеек
- Автоматический перенос строк
- SnapKit для автолэйаута
